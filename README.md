#### 提交示例
执行如下命令
```
git subtree push --prefix [需要提交的子目录的路径] [远程库的源名称] [子分支名称]
```
例如：
```
git subtree push --prefix src/assets/data/stylelib/layout-module-config origin style-layout-module-config
```
